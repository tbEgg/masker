## 介绍

学习`Golang`的练手之作，基于[v2ray-core-0.1](https://github.com/v2ray/v2ray-core/tree/v0.1)。已通过本地测试，暂未实际应用。



## 环境

- `Win10`
- `Go 1.15.0`
- `Chrome`插件`SwitchyOmega`



## 用法

```powershell
go build -o node.exe
.\node.exe -config_file client_config.json
.\node.exe -config_file server_config.json
```
